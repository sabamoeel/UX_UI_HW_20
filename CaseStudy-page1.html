CASE STUDY: PARENTZAPP
SUMMARY

A turnkey solution for parents looking to sign their kids up for extra curricular activities

APP OVERVIEW

Similar to other booking search apps like YELP, with added features such as a calendar of events, and the ability to book activities for your kids within the app.

ROLE + RESPONSIBILITIES

I created this app on my own and am the sole researcher and designer.

DELIVERABLES

The deliverables for this project are User Interviews, a lo-fi wireframe and a clickable proto.

TOOLS USED

Figma, Figjam, Google Docs, iPhone

PHASE 1: USER RESEARCH

For this project, I conducted 6 user interviews with Bay Area moms with small children. https://docs.google.com/document/d/1Wwb9BnhBTvU2jnNj7x9M2nyQUtvx_l_FAC1B39Ed9iQ/edit?usp=drive_link

PHASE 2: DEFINITION

For this project, I conducted 6 user interviews with Bay Area moms with I realized I needed to create an app similar to Yelp so that I can have searches by area and ratings.

PHASE 3: IDEATION

The main function of this app was to create a search function that would allow the user to search a business in their area.

PHASE 5: USER TESTING

Users felt they were able to complete the search and find what they were looking for https://drive.google.com/file/d/1HiW3F1LuBdKT3lz-j1vomdOttDzKsIvd/view?usp=drive_link

I feel that this project has legs and should be iterated over time and completed because it has commercial viablity. The app would be supported with a transaction fee.
